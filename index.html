I created this pneumatic engine based on provided technical drawings, which required a high level of precision throughout the build. I machined the cylinder block from steel, the crank and flywheel from brass, and the piston and connecting rod from aluminum. Avionic Headphone Tester

Electronics 2025

Technical Skills Applied

Circuit design, PCB layout, analog signal processing, op-amp configuration, electronic testing protocols

Project Description

designed and built test equipment for aviation headphone verification and quality assurance. The system features dual channel op-amp signal conditioning, a 10-segment LED bargraph display for real-time audio level monitoring, and integrated fuse protection for equipment safety. I created the initial schematic in Proteus, tested the circuit on a breadboard, and then implemented it on a custom stripboard PCB. During development I added several modifications including a fuse for protection, a power indicator LED, a volume LED bar, and filtering stages to improve signal quality. The final design includes frequency response testing capabilities and automated pass/fail indicators.

Arduino Traffic Light System

×
Embedded 2024

Technical Skills Applied

Embedded systems programming, Arduino IDE, interrupt handling, state machine design, real time control systems, urban infrastructure simulation

Project Description

I developed a traffic control system that simulates pedestrian crossing functionality with dual direction traffic management. I fully coded the system in the Arduino IDE, implemented interrupt driven pedestrian button input, and used timer based state machine logic with safety interlocks to prevent conflicting light states. I built and tested the circuit on a breadboard, refining the design through iterative debugging. The system includes realistic timing sequences such as amber warning phases and a pedestrian countdown display, demonstrating my understanding of embedded systems programming and control applications in urban infrastructure.

System Architecture

1

State Machine Design

Implemented timer-based state machine logic to manage traffic light sequences and pedestrian crossing phases

2

Interrupt System

Configured interrupt-driven pedestrian button input with debouncing and priority handling

3

Safety Interlocks

Programmed safety interlocks to prevent conflicting light states and ensure pedestrian safety

4

Real-time Control

Implemented realistic timing sequences with amber warnings, countdown displays

Project Outcome

Successfully developed an intelligent traffic control system that accurately simulates real-world urban infrastructure operations. The project demonstrates comprehensive understanding of embedded systems programming, real-time control principles, and safety-critical system design. The completed system provides reliable traffic management with full pedestrian integration and emergency response capabilities, showcasing practical applications of embedded systems in urban infrastructure.

Python Stock Predictor

×
ML/AI 2024

Technical Skills Applied

Machine learning, deep learning, GRU neural networks, Python programming, financial analysis, API integration, data visualisation, risk management

Project Description

I built a financial prediction system that combines deep learning neural networks with technical analysis indicators. The model uses a Gated Recurrent Unit (GRU) architecture for time series forecasting, trained on historical stock data collected through the Alpha Vantage API. I implemented Bollinger Bands for volatility analysis, applied sliding window data preprocessing, and designed custom loss functions to optimise prediction performance. The system achieved 63% directional accuracy on validation data and includes integrated risk management metrics with a visualisation dashboard for tracking prediction outputs.
